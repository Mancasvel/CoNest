## Visión y Propuesta de Valor

CoNest surge como una idea revolucionaria en 2024, desarrollada por un grupo de estudiantes innovadores de la Universidad de Sevilla que identificaron dos realidades sociales urgentes: la dificultad de acceso a vivienda asequible para estudiantes y la soledad no deseada que experimentan muchas personas mayores.

Nuestra misión es transformar el modo en que entendemos la vivienda y la conexión intergeneracional, creando comunidades más inclusivas y solidarias. Lo que comenzó como un proyecto académico ambicioso pronto capturó la atención de inversores y colaboradores que vieron su enorme potencial.

## Modelo de Negocio

### Para Personas Mayores (Anfitriones)
- **Beneficios económicos**: Ingresos adicionales por compartir espacio en sus hogares
- **Compañía y seguridad**: Reducción de la soledad y sensación de seguridad al tener a alguien en casa
- **Apoyo cotidiano**: Asistencia en tareas digitales, pequeñas compras y gestiones diarias
- **Intercambio generacional**: Oportunidad de compartir experiencias y aprender de las nuevas generaciones

### Para Estudiantes
- **Alojamiento asequible**: Acceso a vivienda con costos significativamente menores que las opciones tradicionales
- **Ubicaciones privilegiadas**: Posibilidad de vivir en zonas cercanas a campus universitarios o centros urbanos
- **Entorno familiar**: Ambiente propicio para el estudio y desarrollo personal
- **Aprendizaje intergeneracional**: Contacto con experiencias y conocimientos de personas mayores

### Proceso y Funcionamiento
1. **Registro y Verificación**: Riguroso proceso de verificación para ambas partes (verificación de identidad, académica y antecedentes)
2. **Emparejamiento Inteligente**: Sistema basado en preferencias, intereses compatibles y expectativas de convivencia
3. **Entrevistas y Conocimiento**: Facilitación de reuniones previas para confirmar la compatibilidad
4. **Acuerdo Formal**: Documento detallado con condiciones, responsabilidades y normas de convivencia
5. **Seguimiento y Soporte**: Acompañamiento continuo durante toda la convivencia

### Modelo de Ingresos
- **Comisión por colocación**: Porcentaje del 10% sobre el alquiler mensual
- **Servicios de verificación premium**: Opciones adicionales de verificación y filtrado
- **Membresías anuales**: Para acceso a funcionalidades avanzadas de la plataforma
- **Servicios complementarios**: Mediación, asesoría legal, seguros de convivencia

## Impacto Social

CoNest está alineado estratégicamente con múltiples Objetivos de Desarrollo Sostenible (ODS) de la Agenda 2030:

- **ODS 3: Salud y Bienestar**: Reducción de la soledad en mayores, promoviendo la mejora significativa de su bienestar físico y mental. Estudios demuestran que la compañía regular reduce hasta un 40% el riesgo de deterioro cognitivo en adultos mayores.

- **ODS 10: Reducción de las Desigualdades**: Proporcionamos acceso a vivienda asequible para jóvenes con menos recursos económicos. En ciudades universitarias, los estudiantes pueden ahorrar hasta un 50% en gastos de alojamiento.

- **ODS 11: Ciudades y Comunidades Sostenibles**: Optimizamos el uso de viviendas existentes, reduciendo la presión sobre el mercado inmobiliario y promoviendo modelos habitacionales más sostenibles.

- **ODS 17: Alianzas para Lograr los Objetivos**: Colaboramos con universidades, administraciones públicas y organizaciones del tercer sector para maximizar nuestro impacto.

### Indicadores de Impacto

- Reducción de hasta un 30% en sentimientos de soledad en mayores participantes
- Ahorro promedio del 50% en costos de alojamiento para estudiantes
- Creación de nuevas plazas de alojamiento asequible sin necesidad de nueva construcción
- Mejora de relaciones intergeneracionales en las comunidades donde operamos

## Arquitectura Técnica

CoNest está construido con tecnologías modernas para garantizar escalabilidad, seguridad y experiencia de usuario excepcional:

### Frontend
- **Framework**: Next.js 14 (App Router) con enfoque en renderizado híbrido (SSR, SSG, CSR)
- **UI/UX**: TailwindCSS con componentes personalizados y biblioteca HeroUI
- **Interactividad**: React Hooks y Context API para gestión de estado
- **Experiencia**: Diseño responsive con animaciones fluidas y transiciones optimizadas
- **Accesibilidad**: Cumplimiento de estándares WCAG 2.1 para interfaces inclusivas

### Backend
- **Infraestructura**: Next.js API Routes para endpoints serverless
- **Base de Datos**: PostgreSQL gestionada por Supabase
- **Autenticación**: Sistema OAuth y basado en cookies mediante Supabase Auth
- **Almacenamiento**: Supabase Storage para imágenes de perfil y documentos
- **Comunicación**: API RESTful con TypeScript para tipado estricto

### Integraciones
- **Pagos**: Stripe para procesamiento seguro de transacciones
- **Comunicación**: Sistema de chat en tiempo real integrado
- **Geolocalización**: Mapas interactivos para búsqueda de alojamiento
- **Verificación**: Sistemas externos para validación de identidad y antecedentes
- **IA**: Algoritmos de emparejamiento basados en compatibilidad real

### Despliegue y DevOps
- **CI/CD**: Pipelines automatizados en Vercel para integración y despliegue continuo
- **Monitorización**: Análisis de rendimiento y errores en tiempo real
- **Seguridad**: Protocolos avanzados de cifrado y protección de datos personales
- **Escalabilidad**: Arquitectura serverless para adaptarse a picos de demanda

### Características Técnicas Destacadas

1. **Chatbot Integrado**: Asistente virtual accesible desde cualquier página para resolución inmediata de dudas sobre el proceso o la plataforma.

2. **Sistema de Perfiles**: Gestión avanzada de perfiles con algoritmo de coincidencia basado en compatibilidad real (intereses, horarios, preferencias).

3. **Diseño Responsive**: Experiencia de usuario optimizada para dispositivos móviles, tablets y escritorio con interfaces adaptativas.

4. **Verificación Multi-nivel**: Sistema de verificación de identidad, académica y de seguridad con flujos específicos para cada tipo de usuario.

5. **Chat Seguro**: Comunicación privada entre usuarios antes de compartir datos personales, facilitando el conocimiento previo.

6. **Panel Administrativo**: Herramientas de gestión para seguimiento de usuarios, resolución de conflictos y analítica avanzada.
